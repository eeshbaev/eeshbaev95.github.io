# App link verification files

Generated by `npm run well-known:generate` (requires `APPLE_TEAM_ID` and `ANDROID_SHA256_FINGERPRINT` in `.env`).

After generation, run `npm run export:web` — Expo copies `public/` into the static export root so hosts serve:

- `https://ochiqmuloqot.uz/.well-known/apple-app-site-association`
- `https://ochiqmuloqot.uz/.well-known/assetlinks.json`

Do not commit real team IDs or keystore fingerprints. Generate locally before each production web deploy.

See [docs/SHARING.md](../docs/SHARING.md) for the full deep-link checklist.
